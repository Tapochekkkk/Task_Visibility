# Test_UI

Visibility manager

Демонстративный проект функции Interact в UE5.6 на Blueprints.

Управление: WASD - передвижение ЛКМ/ПКМ - управление камерой

Использование: На карте уже расставлены объекты. При нажатии на кнопку в UI меняется видимость этих объектов.

BP_Manager:
Name - имя объекта
Tag - тэг объекта

Требования: Наличие UE5.6 FAQ

Зачем вы разработали этот проект?

Выполнение тестового задания

Задача: Создать уровень с несколькими объектами. Реализовать простой UI, с помощью которого пользователь сможет включать и отключать отображение каждого из объектов и всех сразу.
